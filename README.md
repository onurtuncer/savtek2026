# Türkiye Savunma ve Havacılık Sanayi — Master Veri Seti (2000–2025)

Araştırma: AI-destekli derleme (SASAD, SSB, TİM, SIPRI, TÜBİTAK, YÖK, TOBB kaynakları)

## Dosyalar

| Dosya | İçerik | Yıllar |
|---|---|---|
| `1_savunma_sektoru.csv` | İhracat, ciro, AR-GE, istihdam, proje, yerlileşme | 2000–2025 |
| `2_makro_ekonomi.csv` | GSYİH, savunma bütçesi, silah ithalatı payı, kur, enflasyon | 2000–2025 |
| `3_arge_teknoloji.csv` | Toplam AR-GE/GSYİH, TZE personel, savunma AR-GE, merkez sayısı | 2002–2024 |
| `4_yuksekogretim.csv` | Üniversite sayısı, öğrenci, brüt okullaşma, öğretim elemanı | 2000–2024 |
| `5_jeopolitik.csv` | Küresel çatışma sayısı, savunma harcaması, TSK operasyonları | 2000–2025 |
| `6_istihdam_profili.csv` | Mühendis / teknisyen / idari kırılımı | 2008–2024 |
| `7_istihdam_yas_dagilimi.csv` | Yaş grubu tahmini (kestiri) | 2023 kesit |

## Notlar

- **Para birimi**: USD (aksi belirtilmedikçe)
- **tahmini = 1**: Enterpolasyon veya tahmini değer; raporlu değil
- **Boş hücre**: Veri mevcut değil
- **M$**: Milyon USD, **Mr$**: Milyar USD
- **Yüzde sütunları**: % işareti olmadan (örn. 80 = %80)
